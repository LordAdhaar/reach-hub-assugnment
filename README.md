## How to run locally


1. Clone the repo
   ```sh
   git@github.com:LordAdhaar/reach-hub-assugnment.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build an app
   ```sh
   npm start
   ```
4. Go to browser and go to 
    ```sh
    http://localhost:3000/
   ```
